# 🎯 Desafio DIO – Extraindo Insights do Feedback de Clientes Bancários

## 📌 Sobre o desafio
Este repositório foi criado para armazenar a entrega do desafio proposto pela DIO com foco na construção de prompts utilizando Inteligência Artificial.

O objetivo foi desenvolver um prompt estruturado para orientar uma IA na análise de feedbacks de clientes bancários, transformando comentários em insights úteis para tomada de decisão.

---

## 🧠 Objetivo da atividade
Criar um prompt capaz de:

- Analisar feedbacks de clientes bancários;
- Identificar padrões e temas recorrentes;
- Classificar sentimento e urgência;
- Gerar insights acionáveis;
- Respeitar boas práticas de análise e privacidade de dados.
